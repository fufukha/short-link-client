# short-link-client
Simple mock app that for URL shortening.

Created with CodeSandbox
[Go to CodeSandbox](https://codesandbox.io/s/flamboyant-mclaren-qukd7)

## Setup
To run this project, install it locally using yarn:
```
$ cd ../short-link-client
$ yarn install
$ yarn start
```
